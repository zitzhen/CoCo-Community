# 评论API请求及需求手册
## API简介
本API要完成发送Essay评论的需求，通过传输用户Cookie确定用户身份。评论文章，评论内容由POST传入。
## 请求方法
| 请求方法 | 是否可用  |
| ---- | ----- |
| GET  | 不可用   |
| POST | 可用，唯一 |
## 需要的参数
| 参数名     | 类型  | 强制  |
| ------- | --- | --- |
| EssayID | 数字  | 是   |
| content | 字符串 | 是   |
此外，请求体中需要携带Cookie中的token参数
## 数据表结构
| id    | username                           | content     | time  | ip       | essayid          |
| ----- | ---------------------------------- | ----------- | ----- | -------- | ---------------- |
| 数据表ID | Github的用户名，通过传输的token然后githubAPI查询 | POST传入的评论内容 | 请求的时间 | 客户端的IP地址 | POST传入的评论EssayID |
